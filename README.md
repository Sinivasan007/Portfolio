# Sinivasan S — Portfolio

A 3D interactive portfolio website with parallax scrolling, smooth animations, and light/dark theme — built to showcase my work as a Full Stack Developer.

🔗 **Live site:** https://sinivasan007.github.io/Portfolio/

## Features

- Interactive 3D background built with Three.js (rotating geometry, particles, mouse + scroll parallax)
- Smooth scroll-triggered reveal animations powered by GSAP & ScrollTrigger
- Light / dark theme toggle with saved preference
- Fully responsive layout for desktop, tablet, and mobile
- Sections for About, Skills, Projects, Experience, and Contact
- Working contact form and one-click downloadable résumé

## Tech Stack

- HTML5, CSS3 (custom properties for theming)
- JavaScript (ES6+)
- [Three.js](https://threejs.org/) — 3D rendering
- [GSAP](https://gsap.com/) + ScrollTrigger — animation & parallax

## Sections

| Section | Description |
|---|---|
| Hero | Intro with animated headline and call-to-action |
| About | Background, education, and certification |

| Projects | Mentor-Mentee Platform, FitShop, AI Landing Page Generator, CRUD App |
| Experience | Priga Solutions & Kevell Guru internships |
| Contact | Contact form, social links, and résumé download |

## Run Locally

This is a single static HTML file — no build step required.

```bash
git clone https://github.com/Sinivasan007/Portfolio.git
cd Portfolio
```

Then simply open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
